It is the key

Great, the website's running but what does it mean if you can't stay up to date? A simple SMTP can go a long way to keep you informed. Ehlo, starttls, login and the smtplib library are a must and don't forget to make your life easier by instantiating the email object with the email library so that you can personalize your messages. Once all of this is done, just don't forget to authorize the access to your e-mail!

In what concerns the Twilio app, you'll find that their documentation is surprisingly guiding, to the point where they set you up pretty nicely for their API call.

You can find my code right here! And checkout the Twilio documentation for more information.

Now you only have to answer the real question: what information?
