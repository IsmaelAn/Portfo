Building a real estate tracker

It has to come from somewhere and that's where the scraping comes in handy. While there are many third-party softwares readily available to such an occasion, I found it more pedagogic to write my own. Beautiful Soup can perfectly accomplish this and using the standard html.parser got me almost everything I needed. In fact, some parts of the process had to be complemented with Selenium as the interaction with some websites was tricky...

As for the data warehousing, sqlalchemy came to my rescue. For the sake of speed and ease, this ORM library spares me the time of having to write all my SQL queries and programming a class to instantiate each database entry is all we need to make this chosen methodology work, not to mention the security issues that are quickly handled by using it.

Once the database is created, we only have to call for the handiness of Jinja and develop some neat HTML and CSS to present our findings. Of course, some data science visualization libraries like Seaborn and Bokeh are always welcome here.
